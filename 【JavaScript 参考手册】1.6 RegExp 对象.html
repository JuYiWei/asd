1.7 RegExp 对象

  • 语法
  • 修饰符
  • 正则规则
    • 范围
    • 元字符
    • 量词
  • 对象方法
  • String 方法

语法

var patt = new RegExp(pattern,modifiers);
var patt = /pattern/modifiers;

pattern     （模式） 描述了表达式的模式
modifiers   (修饰符) 用于指定全局匹配、区分大小写的匹配和多行匹配

修饰符

i   执行对大小写不敏感的匹配。
g   执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）。
m   执行多行匹配。

正则规则

范围

    reg = /[a-c]/g      // a-c
    reg = /[^a-c]/g     // 非 a-c

    reg = /[ac]/g       // a||c
    reg = /[^ac]/g      // 非 （a||c）

    reg = /[^a-c][^1]/g // （非 a-c） &&（非 1）
    reg = /[a-c][1]/g   // （a-c）&& （1）
    reg = /[a-c]|[1]/g  // （a-c） || （1）

    reg = /a|c/g        // a || c
    reg = /ac/g         // ac 完整字符串

元字符

\w      单词字符 a-z、A-Z、0-9、下划线
\W      非单词字符

\d      数字
\D      非数字

\s      空白字符
\S      非空白

\b      单词边界
\B      非单词边界

\0      NULL 字符
\n      换行
\f      换页
\r      回车
\t      制表符
\v      垂直制表符
\xxx    八进制数 xxx 规定是字符
\xdd    十六进制 dd 规定的字符
\uxxxx  十六进制 xxxx 规定的 unicode 字符

量词

n+      至少一个 n
n*      零个或多个 n
n?      零个或一个 n
n{X}    X 个 n
n{X,}   n 连续出现至少 X 次
n{X,Y}  n 连续出现至少 X 次，至多 Y 次

n$      结尾为 n
^n      开头为 n
?=n     其后紧接指定字符串 n
?!n     其后没有紧接指定字符串 n

对象方法

compile 编译正则表达式
exec    检索字符串中指定的值。返回找到的值，并确定其位置
test    检索字符串中指定的值。返回 true 或 false

String 方法

search  检索与正则表达式相匹配的值
match   找到一个或多个正则表达式的匹配
replace 替换与正则表达式匹配的子串
split   把字符串分割为字符串数组
