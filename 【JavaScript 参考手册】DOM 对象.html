3.1 Document 对象

  • 文档信息
  • 元素集合
  • 修改文档
  • 其他

文档信息

document.title      返回当前文档的标题。
document.URL        返回文档完整的URL
document.baseURI    返回文档的绝对基础 URI
document.doctype    返回与文档相关的文档类型声明
document.domain     返回当前文档的域名
document.readyState 返回文档状态
document.referrer   返回载入当前文档的文档的 URL

document.documentMode   返回用于通过浏览器渲染文档的模式
document.documentURI    设置或返回文档的位置
document.domConfig      返回 normalizeDocument() 被调用时所使用的配置
document.inputEncoding  返回用于文档的编码方式
document.lastModified   返回文档被最后修改的日期和时间

元素集合

document.links      返回对文档中所有 Area 和 Link 对象引用
document.anchors    返回对文档中所有 Anchor 对象的引用
document.applets    返回对文档中所有 Applet 对象的引用
document.images     返回对文档中所有 Image 对象引用
document.forms      返回对文档中所有 Form 对象引用
document.scripts    返回对文档中所有 脚本 对象引用
document.embeds     返回对文档中所有 嵌入的内容 对象引用

document.getElementById()           返回对拥有指定 id 的第一个对象的引用
document.getElementsByName()        返回带有指定名称的对象集合
document.getElementsByTagName()     返回带有指定标签名的对象集合
document.getElementsByClassName()   返回文档中所有指定类名的元素集合，作为 NodeList 对象

document.body               返回文档的body元素
document.documentElement    返回文档的根节点
document.activeElement      返回当前获取焦点元素

document.implementation         返回处理该文档的 DOMImplementation 对象
document.querySelectorAll()     返回文档中匹配的CSS选择器的所有元素节点列表

修改文档

document.createAttribute()          创建一个属性节点
document.createComment()            创建注释节点
document.createElement()            创建元素节点
document.createTextNode()           创建文本节点
document.createDocumentFragment()   创建空的 DocumentFragment 对象

document.addEventListener()         向文档添加句柄
document.removeEventListener()      移除文档中的事件句柄(由 addEventListener() 方法添加)

document.renameNode()   重命名元素或者属性节点。

document.write()        向文档写 HTML 表达式 或 JavaScript 代码。
document.writeln()      等同于 write() 方法，不同的是在每个表达式之后写一个换行符。

其他

document.open()     打开一个流，以收集来自任何 document.write() 或 document.writeln() 方法的输出
document.close()    关闭流，并显示选定的数据

document.importNode()           把一个节点从另一个文档复制到该文档以便应用
document.normalize()            删除空文本节点，并连接相邻节点
document.normalizeDocument()    删除空文本节点，并连接相邻节点的

document.cookie                 设置或返回与当前文档有关的所有 cookie。
document.strictErrorChecking    设置或返回是否强制进行错误检查
document.adoptNode(node)        从另外一个文档返回 adapded 节点到当前文档。
