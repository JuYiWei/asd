元素尺寸、位置

element.clientHeight    在页面上返回内容的可视高度（不包括边框，边距或滚动条）
element.clientWidth     在页面上返回内容的可视宽度（不包括边框，边距或滚动条）

element.offsetHeight    返回元素的高度，包括边框和填充，但不是边距
element.offsetWidth     返回元素的宽度，包括边框和填充，但不是边距

element.offsetParent    返回元素的偏移容器
element.offsetLeft      返回当前元素的相对水平偏移位置的偏移容器
element.offsetTop       返回当前元素的相对垂直偏移位置的偏移容器

element.scrollHeight    返回整个元素的高度（包括带滚动条的隐蔽的地方）
element.scrollWidth     返回元素的整个宽度（包括带滚动条的隐蔽的地方）

element.scrollLeft      返回当前视图中的实际元素的左边缘和左边缘之间的距离
element.scrollTop       返回当前视图中的实际元素的顶部边缘和顶部边缘之间的距离

相关元素

element.ownerDocument   返回元素的根元素
element.parentNode      返回元素的父节点
element.firstChild      返回元素的第一个子节点
element.lastChild       返回的最后一个子元素
element.nextSibling     返回该元素紧跟的一个节点
element.previousSibling 返回某个元素紧接之前元素
element.querySelector() 返回匹配指定 CSS 选择器元素的第一个子元素

element.childNodes                  返回元素的一个子节点的数组
element.getElementsByTagName()      返回指定标签名的所有子元素集合。
element.getElementsByClassName()    返回文档中所有指定类名的元素集合，作为 NodeList 对象。

结点操作

element.cloneNode()         克隆某个元素
element.appendChild()       添加一个新的子元素
element.insertBefore()      现有子元素之前插入一个新的子元素
element.removeChild()       删除一个子元素
element.replaceChild()      替换一个子元素

element.addEventListener()      向指定元素添加事件句柄
element.removeEventListener()   移除由 addEventListener() 方法添加的事件句柄
element.focus()                 设置文档或元素获取焦点

属性内容

element.id                  设置或返回元素的 id
element.title               设置或返回元素的 title
element.textContent         设置或返回元素的 文本内容
element.lang                设置或返回元素的 语言
element.accessKey           设置或返回元素的 accesskey（快捷键）
element.contentEditable     设置或返回元素的 内容是否可编辑
element.dir                 设置或返回元素的 文本方向
element.style               设置或返回元素的 样式
element.className           设置或返回元素的 class
element.tabIndex            设置或返回元素的标签顺序。
element.innerHTML           设置或返回元素的内容。


element.nodeName        返回元素的标记名（大写）
element.nodeType        返回元素的节点类型
element.nodeValue       返回元素的节点值
element.namespaceURI    返回元素的命名空间的 URI。
element.tagName         返回元素的标记名（大写）

element.classlist           返回元素的类名数组
element.attributes          返回元素的属性数组
element.getAttribute()      返回指定元素的属性值
element.getAttributeNode()  返回指定属性节点

属性操作

element.removeAttribute()       删除指定的属性
element.removeAttributeNode()   删除指定属性节点

element.setAttribute()      设置或者改变指定属性并指定值
element.setAttributeNode()  设置或者改变指定属性节点
element.setIdAttribute()
element.setIdAttributeNode()

判断与比较

element.hasAttribute()  检查元素中是否存在指定的属性
element.hasAttributes() 检查元素是否存在任何属性
element.hasChildNodes() 检查元素是否存在任何子元素
element.hasFocus()      检测文档或元素是否获取焦点
element.isEqualNode()   检查两个元素是否相等
element.isSameNode()    检查两个元素所有有相同节点
element.isSupported()   检查元素是否存在指定特征

element.isContentEditable       检查元素内容是否可编辑返
element.isDefaultNamespace()    检查是否指定了namespaceURI

element.compareDocumentPosition()   比较两个元素的文档位置

其他

element.getFeature()    返回指定特征的执行 APIs 对象。

element.getUserData()   返回元素中关联键值的对象。
element.setUserData()   在元素中为指定键值关联对象。

nodelist.item()     返回元素基于文档树的索引
nodelist.length     返回节点列表的节点数目
element.toString()  一个元素转换成字符串
element.normalize() 使得此成为一个"normal"的形式，其中只有结构，隔开Text节点，即元素（包括属性）下面的所有文本节点，既没有相邻的文本节点也没有空的文本节点
