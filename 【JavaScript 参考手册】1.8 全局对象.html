1.8 全局属性

  • 全局属性
  • 全局函数
    • 编码
    • 检查
    • 强制转换

全局属性

Infinity    代表正的无穷大的数值。
NaN         指示某个值是不是数字值。
undefined   指示未定义的值。

全局函数

编码

  • URL
    var url = 'www.baidu.com?a=1&b=2';
    var res;
    
    res = encodeURIComponent(url);
    res = decodeURIComponent(res);
  • URI
    res = encodeURI(url);
    res = decodeURI(res)

检查

    res = isFinite(123);// 有限数字
    res = isNaN(123);// 不是数

强制转换

    res = Number('1.23');
    res = parseFloat('1.23');
    res = parseInt('1.23');
    
    res = String(1.23);

其他

  • 执行文字脚本
    eval("x=10;y=20;document.write(x*y)");
